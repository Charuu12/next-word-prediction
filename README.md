# Next Word Prediction using LSTM

This project builds a deep learning model to predict the next word in a sentence using an LSTM network.

## Problem

Predicting the next word helps in improving text-based applications like autocomplete and text generation by understanding word sequences.

## Solution

An LSTM-based model is trained on text data after preprocessing and tokenization to learn patterns between words and generate accurate next-word predictions.

## Key Features

* Text preprocessing and tokenization
* Sequence generation for training
* LSTM-based prediction model
* Interactive Streamlit interface

## Tech Stack

* Python
* Keras
* NumPy, Pandas
* Streamlit
